# My Resume
## Introduction

项目基于 [DIYGOD](https://github.com/DIYgod/Resume)

## Usage

1. Star 本项目
1. Clone 到本地
1. 在 `index.html` 和 `print.html` 文件中填写你的信息
1. 生成PDF
1. 部署上线
1. Enjoy it!

## Development

```
$ npm install
$ npm install -g gulp
$ gulp
```
